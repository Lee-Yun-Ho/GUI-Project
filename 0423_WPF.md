## 데이터 바인딩

* 데이터 바인딩이란 컨트롤과 엘리먼트를 데이터에 연결시키는 기술이다. 
* 컨트롤은 데이터를 사용자에게 보여주는 것과 사용자가 데이터를 변경할 수 있게 해주는 두 가지 기능을 제공

```xaml
<Window x:Class="WPF_DataBinding.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:d="http://schemas.microsoft.com/expression/blend/2008"
        xmlns:mc="http://schemas.openxmlformats.org/markup-compatibility/2006"
        xmlns:local="clr-namespace:WPF_DataBinding"
        mc:Ignorable="d"
        Title="라벨, 스크롤바 데이터 바인딩" Height="350" Width="525">
    
    <StackPanel >
        
        <!--소스에서 타겟으로-->
        <!--모드를 TwoWay 바인딩으로 설정할 경우 양방향 데이터 갱신-->

        <!-- Source. -->
        <ScrollBar Name="scrollbar"
               Orientation="Horizontal" Margin="24"
               Maximum="100" LargeChange="10" SmallChange="1" />

        <!-- Target. -->
        <!--바인딩 정의가 있는 컨트롤은 항상 타겟이다.-->
        <Label HorizontalAlignment="Center"
           Content="{Binding ElementName=scrollbar, Path=Value, Mode=OneWay}" />

    </StackPanel>
    
</Window>
```
